Project Overview
This repository contains my MLOps learning journey from Day 9 to Day 22.
Each day contains a practical task/project focused on real-world MLOps concepts.
The goal is to learn end-to-end ML lifecycle: training, deployment, monitoring, and automation.

--------------------------------------------------

--------------------------------------------------

How to Use
1. Clone the repository:
git clone <repo-url>

2. Go to the specific day folder:
cd Day10_Model_Evaluation_Tuning

3. Install dependencies:
pip install -r requirements.txt
(If requirements file is not present, install required libraries manually)

4. Run the project:
python <script_name>.py
(or run the notebook if provided)
