# Day 9 – Model Building (Linear Regression)

## Objective
To build a basic regression model and evaluate its performance.

## Steps Performed
- Loaded and prepared dataset
- Split data into training and testing sets
- Built Linear Regression model
- Evaluated model using MAE, MSE, RMSE, and R² score

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Outcome
Successfully trained a regression model and understood basic model evaluation metrics.
