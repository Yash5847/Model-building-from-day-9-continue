# BigMart Sales Prediction

This project predicts sales for BigMart outlets using a trained Machine Learning model.

## Features Used
- Item_Identifier
- Item_Weight
- Item_Fat_Content
- Item_Visibility
- Item_MRP
- Item_Type
- Outlet_Identifier
- Outlet_Establishment_Year
- Outlet_Size
- Outlet_Location_Type
- Outlet_Type

## Technologies
- Python, Pandas, Scikit-learn
- Linear Regression Model with Pipeline (preprocessing + imputation)
- Streamlit for interactive deployment
